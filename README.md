# 📡 Network Monitoring Stack

Production-ready monitoring and DNS infrastructure using Docker.

## 🚀 Overview
This project provides a complete monitoring stack using:

- Zabbix 6.4 for infrastructure monitoring  
- Grafana for visualization and dashboards  
- Pi-hole for DNS filtering  
- Unbound for recursive DNS with DNSSEC  

## 🧱 Architecture
- Containerized using Docker Compose  
- Modular services for monitoring and DNS  
- Designed for ISP / NOC environments  

## ⚙️ Technologies
- Docker / Docker Compose  
- Zabbix  
- Grafana  
- Pi-hole  
- Unbound DNS
